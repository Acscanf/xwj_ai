## shrink 收缩计算
- 溢出空间 300px 让子元素来shrink 分摊
- 分摊的比例？
    flex-shrink： 2:1   500:400

- 分摊权重
    总的权重？
    500 * 2 + 400 * 1 = 1400
    1000/1400 = 0.7142857142857143  71.43%
    400/1400 = 0.28571428571428577  28.57%


    500 - 300*71.43% = 285.71px
    400 - 300*28.57% = 314.29px

## grow
    600 - 300 - 200 = 100 剩余空间
    flex-grow  1:2
    100 * 1 / 3 = 33.33px
    100 * 2 / 3 = 66.66px
